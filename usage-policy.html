이용 정책
시행일: 2025년 3월 1일 | 최종 수정: 2026년 3월 1일

본 이용 정책(이하 "정책")은 BRAVECAT이 운영하는 LEVEL UP: REBOOT 서비스(이하 "서비스")의 올바른 이용 기준을 정합니다. 서비스를 이용함으로써 본 정책에 동의하는 것으로 간주됩니다.

1. 기본 이용 원칙

LEVEL UP: REBOOT는 사용자의 자기계발과 건강한 생활 습관 형성을 돕기 위한 서비스입니다. 모든 사용자는 다음 원칙을 준수해야 합니다:

  • 정직하고 성실하게 서비스를 이용할 것
  • 다른 사용자의 권리를 존중할 것
  • 서비스의 공정한 운영에 기여할 것
  • 관련 법률 및 규정을 준수할 것

2. 금지되는 행위

2.1 데이터 조작 및 부정 행위

다음 행위는 엄격히 금지됩니다:

  • 퀘스트 완료 데이터를 허위로 기록하는 행위
  • 자동화 도구(봇, 매크로, 스크립트 등)를 사용하여 퀘스트를 완료하거나 포인트를 획득하는 행위
  • Google Fit 데이터를 조작하여 허위 운동 기록을 전송하는 행위
  • GPS 위치 정보를 위조(GPS 스푸핑)하는 행위
  • 레이드/던전 참여를 부정하게 조작하는 행위
  • 계정 간 포인트 또는 데이터를 부정하게 이전하는 행위

2.2 커뮤니티 행위 기준

  • 다른 사용자에 대한 괴롭힘, 비방, 차별적 언행
  • 불쾌하거나 유해하거나 공격적인 프로필 이미지 또는 닉네임 사용
  • 스팸, 광고, 또는 상업적 목적의 메시지 전송
  • 타인의 개인정보를 무단으로 공개하거나 수집하는 행위
  • 다른 사용자를 사칭하는 행위

2.3 기술적 남용

  • 서비스의 보안 시스템을 우회하거나 무력화하는 행위
  • 서비스 서버에 과도한 부하를 발생시키는 행위
  • 서비스의 소스 코드를 무단으로 역설계, 디컴파일, 분해하는 행위
  • 서비스의 API나 데이터베이스에 무단으로 접근하는 행위
  • 다중 계정을 생성하여 서비스를 남용하는 행위

3. 소셜 기능 이용 기준

3.1 글로벌 랭킹

랭킹은 공정한 경쟁을 위해 운영됩니다. 부정 행위를 통해 획득한 점수는 삭제될 수 있으며, 반복적인 위반 시 랭킹에서 영구 제외될 수 있습니다.

3.2 친구 기능

친구 추가 기능은 상호 존중을 기반으로 이용되어야 합니다. 불필요한 친구 요청을 반복하거나, 친구 기능을 통해 괴롭힘을 하는 행위는 금지됩니다.

3.3 인스타그램 연동

인스타그램 프로필 연동은 사용자의 선택 사항입니다. 타인의 인스타그램 계정을 무단으로 연동하거나, 허위 계정을 연동하는 행위는 금지됩니다.

4. 던전/레이드 이용 기준

던전/레이드(Local Raid)는 특정 시간대(06:00~08:00, 11:30~13:30, 19:00~21:00 KST)에 운영되는 협동 콘텐츠입니다.

  • 레이드 참여는 실제 활동을 기반으로 해야 하며, 허위 참여는 금지됩니다
  • 레이드 데이터를 조작하여 보상을 부정 취득하는 행위는 금지됩니다
  • 다른 참여자의 레이드 경험을 고의로 방해하는 행위는 금지됩니다

5. 위반 시 조치

본 정책을 위반한 사용자에 대해 회사는 다음 조치를 취할 수 있습니다:

  1. 경고: 경미한 위반의 경우, 사전 경고를 발송합니다.
  2. 기능 제한: 특정 기능(소셜, 레이드 등)의 이용을 일시적으로 제한합니다.
  3. 데이터 초기화: 부정하게 획득한 포인트, 스탯, 랭킹을 초기화합니다.
  4. 계정 정지: 심각한 위반의 경우, 일정 기간 계정을 정지합니다.
  5. 영구 차단: 반복적 위반 또는 중대한 위반의 경우, 계정을 영구 삭제합니다.
