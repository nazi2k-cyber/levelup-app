관리자 인증 방식

관리자 권한은 Firebase Custom Claims로 관리됩니다.

  • master: true — 마스터 계정 (모든 권한 + 권한 관리)
  • admin: true — 관리자 (관리자 페이지 전체 접근)
  • adminOperator: true — 운영자 (관리자 페이지 운영 권한, 마스터가 부여)
