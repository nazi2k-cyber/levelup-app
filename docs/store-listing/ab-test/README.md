# Store Listing A/B Test — Phase 2 전환율 개선

## 목적
설치 전환율(CVR) 개선을 위해 스토어 리스팅 텍스트를 3안으로 운영합니다.
Google Play Console의 "Store listing experiments" 기능을 활용하여 실험합니다.

## 변형 안 요약

| 변형 | 컨셉 | 타겟 키워드 | 톤 |
|------|------|-------------|-----|
| **A (Control)** | RPG 모험 강조 | RPG, Quest, Level Up | 게이머 감성 |
| **B** | 습관 형성 · 자기계발 강조 | 습관, 자기계발, 목표달성 | 실용/코칭 |
| **C** | 소셜 경쟁 · 커뮤니티 강조 | 랭킹, 경쟁, 친구, 레이드 | 소셜/도전 |

## 측정 지표
- **Primary:** 설치 전환율 (CVR = Install / Store View)
- **Secondary:** 첫 세션율, D1 리텐션
- **실험 기간:** 최소 14일 (유의수준 95%)

## 운영 방법
1. Google Play Console → Store listing experiments
2. Variant A = 현재 리스팅 (대조군)
3. Variant B, C를 실험군으로 등록
4. 트래픽 분배: A 34% / B 33% / C 33%
5. 14일 후 CVR 유의차 확인 → 승자 안 전체 적용
