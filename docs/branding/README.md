# BRAVECAT STUDIOS 로고 시안

이 폴더의 `bravecat-studios-logo.svg`는 **원형 엠블럼 + 고양이 얼굴 중심 구조**의 오리지널 시안입니다.

## 구성
- 더블 링 원형 배지
- 중앙 기하학적 고양이 얼굴
- 상단 브랜드명 `BRAVECAT STUDIOS`
- 하단 슬로건 `CREATE • PLAY • ROAR`
- 기본 컬러
  - Deep Green: `#0B5A44`
  - Cream: `#F6F3E9`

## 사용 가이드
- 앱 아이콘/프로필: 텍스트를 제거한 심볼 버전 사용 권장
- 인쇄물: SVG 원본 유지 후 벡터 기반으로 출력
- 상표 리스크 최소화를 위해, 실제 상용화 단계에서 귀/눈/타이포 비율을 추가 커스터마이징 권장
